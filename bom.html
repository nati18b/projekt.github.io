BOM (Browser Object Model) består av objektnavigatorn, historik, skärm, plats och dokument som är fönsterbarn.

BOM behandlar webbläsarkomponenter bortsett från dokumentet, som historik, plats, navigator och skärm (samt vissa andra som varierar beroende på webbläsare). I dokumentnoden finns DOM (Document Object Model), dokumentobjektmodellen, som representerar innehållet på sidan (dokumentet och alla personer som kan göra det, händelser). DOM är Document Object Model, som behandlar dokumentet, själva HTML-elementen. Mannen kan manipulera det med javascript. Noden är den minsta enheten i DOM-trädet eller noderna komponerar DOM-trädet.
